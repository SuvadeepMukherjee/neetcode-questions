# Math (Anki Notes)

Anki Tag ➡️ 03-math

**Q1**: What type is the result of division in Python (using `/`)?

**Solution**:

Always a float.

Example: `y / x` → `2.0` even if both are integers.

**Q2**: For addition , substraction ,multiplication when is the result an integer ? 

**Solution**:

When both operands are integers.

If either operand is a float, the result becomes a float.

**Q3**: What does the `or` keyword do in Python ? 

**Solution**:

- Performs a logical OR operation.
- The expression is `True` if *either* condition is true.

**Q4**:What does the `and` keyword do in Python?

**Solution**:

- Performs a logical AND operation.
- The expression is `True` only if *both* conditions are true.

**Q5**:What does the `not` keyword do in Python?

**Solution**:

- Performs logical negation.
- It flips a boolean: `not True` → `False`.
