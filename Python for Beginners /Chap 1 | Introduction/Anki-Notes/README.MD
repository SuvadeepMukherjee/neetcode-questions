# Introduction (Anki Notes)

**Q1**: How do you include quote characters inside a Python string?

**Solution**:

Use the backslash escape character `\`.

```python
print("They said, \"Hello, world!\"")
```

**Q2**: How do you create a comment in Python ? 

**Solution**:

By using the `#` character.

Everything after `#` on the same line is ignored by Python.
